想和应该

2017年03月27日 / 共 次浏览

由于最近一直在准备考试，所以就没时间更新。

随着昨天初试的考完，心情由考前的焦躁、压抑，想快点结束变成了紧张。

这一点特别奇怪，以前参加各种考试的心情都是考前紧张，考完放松。

而这次考前我一点都不紧张，只是想着早考完早解脱，

但最后一场考完却完全没有解脱放松的感觉，

反而出现了紧张的心情，到不是因为考的不好。

在几个月的复习期间，心情波动比较大，由最初的积极向上，到最后的不坚定，

思前想后，兴趣总不在此，没有多大热情，

可又必须考，没有选择。

可能在这个领域还没有醍醐灌顶吧！

想和应该，中间一条不可跨越的鸿沟，希望将来我的孩子在想和应该之间没有鸿沟吧！

2017-03-27 上午

胡扯
